# DecodeLabs Java Development Tasks

This repository contains the Java programming tasks completed as part of the DecodeLabs Java Development Internship.

## Projects Included

### Task 1: Number Guessing Game
A console-based Java application where the user tries to guess a randomly generated number within a specified range.

#### Features
- Random number generation
- User input handling
- Hint messages (too high/too low)
- Score tracking based on attempts

### Task 2: Student Grade Calculator
A Java application that calculates the total marks, average percentage, and grade of a student based on subject scores.

#### Features
- Accepts marks for multiple subjects
- Calculates total and average
- Assigns grades based on percentage
- Displays complete result summary

### Task 3: ATM Interface
A Java-based ATM simulation developed using Object-Oriented Programming concepts.

#### Features
- Check account balance
- Deposit money
- Withdraw money
- Input validation
- Menu-driven interface

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Control Statements
- Methods and Classes

## How to Run

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Open the desired project folder.

3. Compile the Java file:
   ```bash
   javac FileName.java
   ```

4. Run the program:
   ```bash
   java FileName
   ```
